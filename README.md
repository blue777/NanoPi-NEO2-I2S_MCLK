# Target SBC

- NanoPi-NEO2
- NanoPi-NEO Plus2

# Overview

- Support I2S-MCLK output via PA6 pin.
- Support I2S sampling freq. 384kHz, 768kHz and 1536kHz.
- Bugfix : I2S 24bit issue.
